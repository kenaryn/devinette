# Travaux Pratiques augmentés durant le cours JavaScript avancé

## Fonctionalités
- Message final selon résultat en ajoutant un noeud au DOM
- Réinitialisation
- Refactorisation des comportements communs en function
- Réduction du nombre de variables globales en transmettant les valeurs retournées par les appels de fonctions dans d'autres fonctions
- Désactivation des boutons après chaque tentative (réussie ou ratée)
- Application du principe de responsabilité unique pour chaque fonction
- Chargement de la fonction `init()` selon l'état du DOM en contrôlant la valeur de la propriété `readyState`
- Ajout de JSDoc
